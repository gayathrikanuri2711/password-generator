# 🔐 Smart Password Generator

## 📖 Description

A Python-based application that generates strong and unique passwords using user inputs like username and favourite things (place, food, animal, etc.). It ensures password uniqueness and stores generated passwords securely.

## ⚙️ Features

* Custom password length
* Includes user-based personalization
* Strong password generation (letters, numbers, symbols)
* Prevents duplicate passwords
* Stores passwords using SQLite database

## 🚀 How to Run

```bash
python password_generator.py
```

## 🧰 Technologies Used

* Python
* SQLite
* hashlib
* random & string modules
